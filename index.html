Research on Content Defined Chunking

Keywords: CDC, Rabin, CRC, Irreducible Polynomials, SageMath, Python, Abstract Algebra, git, Github.

Keywords: libsodium, statically generated website, HTML, CSS, Javascript, Markdown, Github.


Porting multisig-hmac

Porting the Javascript implementation of Multisig HMAC to Python and Java. MultiSig HMAC is a novel, symmetic signature scheme allowing a threshold of signers to form a quorum. Wire compatible with normal HMAC, a popular authentication mechanism in web APIs. Extensive testing of all equivalence classes as unit tests. Fully open-source and available on Github, PyPI and MavenCentral:

  • https://github.com/AmalieDue/py-multisig-hmac (PyPI)
  • https://github.com/AmalieDue/java-multisig-hmac(MavenCentral)

Keywords: HMAC, SHA-2, Unit Test, Bit Operations, Python, Java, PyPI, Maven, Git.
